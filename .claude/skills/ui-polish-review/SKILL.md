---
name: ui-polish-review
description: Review ERP pages for modern admin/ERP UI quality, consistency, density, and reuse
---

Review the current UI implementation as if this were a premium ERP product demo.

Check for:

- inconsistent spacing
- weak hierarchy
- poor table ergonomics
- overuse of cards
- missing filters/actions
- unclear status badges
- weak empty/loading/error states
- repeated UI patterns that should be shared
- consumer-app styling that does not fit a serious ERP product

Then:

1. list the most important UI/UX issues
2. prioritize them by impact
3. fix the highest-value issues first
4. preserve the existing architecture where possible

Bias toward:
- dense, professional layouts
- consistency
- better scanability
- better data interaction
- better reuse of shared UI primitives
